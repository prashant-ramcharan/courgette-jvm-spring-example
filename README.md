# Courgette-JVM with Cucumber Spring Example

An example project showing how to use Courgette-JVM with Cucumber Spring.

## System Requirements

* Latest version of Google Chrome
* MacOS / Windows / Linux

## Test Execution

* Option 1: 
    * Run _CourgetteRunner.java_ as a normal JUnit test.

* Option 2: 
    * Execute gradle task 
    ````gradle
        gradle runTest
    ````
